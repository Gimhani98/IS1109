Introduction

Generation in computer terminology is a change in technology a computer is/was being used. Initially, the generation term was used to distinguish between varying hardware technologies. Nowadays, generation includes both hardware and software, which together make up an entire computer system.
The history of the computer goes back several decades and there are five generations since 1945 to today (Some accredited sources mention that six computer generations are available, starting from a zeroth generation ) Each generation is defined by a significant technological breakthrough that changes how computers operate. Decade by decade these breakthroughs are leading computers to more compact, less expensive, but more powerful and efficient machines. However, each generation has brought with it either a new computer or an improved version of an existing computer.
